**Проектный практикум. Семестр 1. "Борьба с растительными оккупантами"**

**Описание проекта:**

Распространение инвазивных видов растений является одной из острых проблем современного мира. 
Все больше и больше территорий захватывается чужеродными для той или иной месности растениями, которые поглащают ресурсы природы и убивают все останые виды.
Предотвратить наступление таких растений гораздо проще, чем бороться с последствиями, поэтому модели для детекции таких растений имеет высокую значимости и очень актулаьны сейчас.
В России наибольшее распространение из инвазивных растений получил Борщевик Сосновского.
В проекте использована модель YOLO и обучение на базе датасета Hogweed dataset и Invasive Plants из Roboflow.

**Основные цели проекта:**
1) Скачать и подготовить к использованию датасеты.
3) Обучить модель на большом датасете для получения модели более высокого качества.
4) Дообучить модель на малом датасете.
5) Протестировать обученную модель на фото инвазивных растений. 

**Структура данных:**
Датасет 1: Общий датасет для инвазивных растений.
Содержание: размеченные фото различных видов инвазивных растений
Формат: YOLOv8
Датасет2: Специализированный датасет (Борщевик Сосновского)
Содержание: рамеченные фото различных видов инвазивных растений
Формат: YOLOv8

**Ключевые этапы проекта:**
1) Подготовка окружения:
   -  Скачивание и установка библиотек: ultralytics, roboflow
   -  Настройка GPU в Google Colab для ускорения вычислений
2) Загрузка данных:
   - Загрузка двух датасетов инвазивных растений из Roboflow
3) Обучение модели YOLOv8:
   - Модель обучается на датасете 1 с параметрами:
     * Количество эпох: 15
     * Размер батча: 16
     * Размер изображения: 640х640
    - Сохранение модели на локальный компьютер в папку загрузок из браузера по умолчанию
4) Дообучение модели на специализированных данных:
   - Модель дообучается на датачет 2 с параметрами:
     * Количество эпох: 15
     * Размер батча: 16
     * Размер изображения: 640х640
    - Сохранение модели на локальный компьютер в папку загрузок из браузера по умолчанию
5) Экспорт моделей
   - Сохранение модели для дальнейшего использования
6) Оценка результатов:  
   - Расчет метрик: Precision,Recall,mAP50 для обоих моделей (обученной и дообученной)
   - Визуализация примеров детекции в отдельном файле

**Состав репозитория:**
Подготовка данных, обучение модели, расчет метрик оценки (ноутбук): Data download_model learning_metrics calcualtion
Сохраненные модели: обученная invasive_detector_best и дообученная hogweed_detector_best
Тестирование модели (ноутбук): Testing models
